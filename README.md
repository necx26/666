# 🚀 股票一键查 (StockMaster AI)

一款基于 WPF 开发的现代化股票行情查看与 AI 分析工具。

### ✨ 功能亮点
- **实时行情**：一键获取最新股价及买卖盘数据。
- **动态图表**：集成分时线与 K 线展示。
- **AI 辅助**：内置智能分析逻辑，助力投资决策。
- **丝滑 UI**：深色模式设计，支持动态交互动画。

### 🛠 技术栈
- .NET 8 / WPF
- LiveCharts (数据可视化)
- 异步网络请求 (HttpClient)

### 📦 如何使用
前往 [Releases](你的链接) 下载最新的 `.exe` 文件即可直接运行。
